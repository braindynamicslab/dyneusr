## **Dynamical Neuroimaging Spatiotemporal Representations**

[DyNeuSR](https://braindynamicslab.github.io/dyneusr/) is a Python visualization library for topological representations of neuroimaging data. 

<p align="center">
	<iframe width="100%" height="500px" seamless frameborder="0" scrolling="no" src="./haxby-tsne/"></iframe>
</p>

DyNeuSR provides an interactive web interface for visualizing and exploring shape graphs. For more examples of this visual interface in action, check out the demos below. For more information about DyNeuSR, check out the [docs](https://braindynamicslab.github.io/dyneusr/).
